# 分页功能完善计划

## 1. 修改 API 函数定义

* **文件**: `src/api/index.ts`

* **修改内容**:

  * 将 `getErrorList` 函数参数改为对象方式接收

  * 添加 `page` 和 `pageSize` 参数，默认值分别为 1 和 10

  * 更新函数返回类型，适配新的后端返回结构

  * 修改请求参数，使用 `page` 而非 `pageIndex`

## 2. 更新 Dashboard 组件

* **文件**: `src/pages/Dashboard/index.tsx`

* **修改内容**:

  * 更新分页状态管理，使用 `page` 而非 `pageIndex`

  * 修改 API 调用方式，使用对象参数

  * 处理新的后端返回结构，提取 `list` 和 `pagination` 信息

  * 更新 `useEffect` 依赖项

  * 调整 Table 组件的 pagination 配置，适配新的分页数据

## 3. 接口参数处理

* 将 `getErrorList` 函数的参数从多个独立参数改为单一对象参数

* 确保参数名与后端保持一致：`page`（当前页码）和 `pageSize`（每页条数）

## 4. 数据处理

* 从后端返回的 `data.list` 中获取错误列表数据

* 从 `data.pagination` 中获取分页信息（total、page、pageSize、totalPages）

* 更新组件状态以反映当前分页情况
